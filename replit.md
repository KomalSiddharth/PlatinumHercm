# Platinum HRCM Dashboard

## Overview
The Platinum HRCM Dashboard is a gamified web application designed to track and enhance personal development across Health, Relationship, Career, and Money (HRCM) life areas. It integrates goal tracking, daily ritual monitoring, and course progress into a single system. Key features include a "Platinum Streak" for consistent activity, real-time feedback, leaderboards, and a progression system to motivate users. The dashboard leverages AI for course recommendations and automated suggestions to address personal challenges, providing a structured and engaging path to personal growth and self-improvement. The project aims to empower users to achieve their personal goals through a gamified and data-driven approach.

## User Preferences
Preferred communication style: Simple, everyday language.

## System Architecture

### UI/UX Decisions
The dashboard features a clean, responsive "New York" style design utilizing shadcn/ui components and the Inter font, with support for both light and dark modes. The primary color is Teal, and the accent is Coral. It includes optimized layouts, a standards-based HRCM rating system scaled out of 10, and a consistent color palette for HRCM cards (Health: Green, Relationship: Purple, Career: Blue, Money: Purple). The application is built with a mobile-first approach, ensuring optimal user experience across various devices through responsive padding, spacing, text sizes, and grid layouts. Interactive elements are touch-friendly, and specific components like the `UnifiedHRCMTable` and `EmotionalTracker` are designed for responsive display.

### Technical Implementations
The frontend is built with React, Vite, Tailwind CSS, TanStack Query, and Wouter for routing. The backend uses Express.js with TypeScript (ES Modules), PostgreSQL with Drizzle ORM, and `connect-pg-simple` for session management. Authentication is session-based with `bcrypt` and implements role-based access control. Data persistence is managed via Drizzle ORM with an `IStorage` interface, Zod validation, and Drizzle Kit for migrations. The system employs an UPSERT logic for saving checkbox states to prevent duplicate database entries and ensures data integrity. A deduplication scoring mechanism selects the most complete weekly data when multiple entries exist. **Platinum Standards Real-Time Sync (Oct 28, 2025)**: When admin adds/updates/deletes platinum standards, the system uses `queryClient.refetchQueries()` to force immediate refetch across all active dashboard users, ensuring instant visibility without page reload. UnifiedHRCMTable implements smart merge logic that preserves existing checked states while adding new standards, with `platinumStandardsData` in useEffect dependencies for automatic updates. **Google Sheets Course Tracking Integration (Oct 29-30, 2025)**: Course Tracking section is powered by Google Sheets API integration. Backend endpoint `/api/courses/tracking` fetches course data from shared Google Sheet (https://docs.google.com/spreadsheets/d/1s3pCuwFC303v2P4PVfZ8V2SizjvUZRVKnpNmUXkShtA/edit?usp=sharing), parses Question-Answer format (Question = Lesson name, Answer = Lesson URL, bold headings/empty answers = Course names), caches for 10 minutes, and serves to all 30K users. **DMP Course Merge - COMPLETE (Oct 30, 2025)**: System automatically merges ALL DMP recording lessons into main "DMP - Daily Magic Practice Recordings" course using advanced merge strategy. Checks: (1) course titles for "recording"/"dmp"/date patterns, (2) individual lesson titles for date patterns (2nd June, 7th June, Sept, etc.), (3) known courses whitelist to skip real courses (Platinum Fast Track, Basic LOA, etc.). **Critical Fix:** Increased Google Sheets fetch range from A1:B1000 to A1:B2000 to capture all 1383 rows. Successfully merges ALL recordings from entire Google Sheet including row 1001 onwards (7th June and beyond) - DMP course now contains 450+ complete lessons covering June, July, Aug, Sept recordings. Section headings like "Sept'25 DMP Recordings", "Platinum DMP" appear as clickable lessons within DMP course (normal behavior - they have URLs in Google Sheet). "June'25 DMP Recordings" and other recording courses removed as standalone course entries. Single source of truth for course management - admin updates Google Sheet and changes automatically reflect across entire platform on next sync.

### Feature Specifications
- **Core Tracking**: Weekly HRCM scoring, daily ritual tracking, comprehensive course progress monitoring with persistent lesson checkboxes, and a Daily Emotional Tracker across 9 time slots. **HRCM Calendar Date Navigation - Exact Match Logic (Oct 29, 2025)**: Calendar navigation works EXACTLY like Daily Emotional Tracker - click calendar icon to select any date, use prev/next buttons to navigate one day at a time, shows selected date in header with "(Today)" label for current day. **Critical Architecture - Exact Date Matching**: Backend `/api/hercm/by-date/:date` uses EXACT date matching (NOT week ranges) - compares `createdAt` date string (YYYY-MM-DD format) with requested date string for perfect match, identical to Emotional Tracker's `/api/emotional-trackers/:date` logic. Only returns data if exact date match found, returns `null` otherwise. Frontend displays data only for dates where user explicitly saved HRCM data. **Read-Only History Mode**: `viewingHistory` state automatically set to true when `selectedDate ≠ today`, triggering complete read-only mode for ALL fields (ratings, problems, feelings, beliefs, actions, platinum standards, assignment checkboxes, checklists). Historical dates display exact snapshot from that specific date - ALL editing disabled to preserve data integrity.
- **Life Problems & Life Skill Map (Oct 30, 2025)**: Dynamically populated Excel-like tabular structure powered by Course Tracking API. Each course becomes a collapsible category, with lessons displayed in two columns: "Life Problem" (lesson name) and "Life Skills" (lesson name as clickable link). Features responsive design with color-matched headers, alternating row colors, and automatic data sync from Google Sheets via course tracking integration. Completely replaced hardcoded data with real-time course library - displays all 40+ courses with 850+ lessons from centralized Google Sheet. URLs are hidden - only lesson names visible as clickable blue links in Life Skills column. **Scrollable Lessons Feature (Oct 30, 2025)**: Each course category's lessons are contained in a scrollable container (`max-h-[400px] overflow-y-auto`) for compact space utilization - courses with many lessons show vertical scrollbar for efficient browsing. **Color Specification (Oct 30, 2025)**: Main heading uses gradient text `bg-gradient-to-r from-primary to-accent bg-clip-text text-transparent` (teal-to-coral gradient) matching Daily Emotional Tracker style. Column headers use distinctive colors - "Life Problem" uses `#bc0000` (dark red) with white text for high visibility, "Life Skills" uses `#006400` (dark green) with white text for matching intensity. Category headings use `text-primary dark:text-accent` (teal in light mode, bright coral in dark mode for visibility). Card background matches Daily Emotional Tracker with gradient `bg-gradient-to-br from-primary/5 to-accent/5 dark:from-primary/10 dark:to-accent/10` and border `border-2 border-primary/30 dark:border-primary/50`. Compact design with minimal padding (p-1.5 sm:p-2), smaller text sizes (text-xs sm:text-sm), and condensed spacing throughout for efficient information display.

Step 4: Server Configuration (Important)
Replit me routing setup karein. Ye aapki technology pe depend karta hai:

For Node.js/Express:
Apne main server file me (usually index.js or server.js):

Copyconst express = require('express');
const app = express();

// Serve static files
app.use(express.static('public'));

// Dashboard route (with authentication)
app.get('/dashboard', requireAuth, (req, res) => {
  res.sendFile(__dirname + '/dashboard.html');
});

// Chatbot route (with authentication)
app.get('/chatbot', requireAuth, (req, res) => {
  res.sendFile(__dirname + '/chatbot.html');
});

app.listen(3000);
For Python/Flask:
Copyfrom flask import Flask, render_template, redirect
app = Flask(__name__)

@app.route('/dashboard')
@login_required
def dashboard():
    return render_template('dashboard.html')

@app.route('/chatbot')
@login_required
def chatbot():
    return render_template('chatbot.html')

Step 5: Authentication Protection
Chatbot page ko protected banane ke liye, page ke top me ye check add karein:

Copy<script>
  // Check if user is logged in
  function checkAuth() {
    // Method 1: Check session/cookie
    const isLoggedIn = document.cookie.includes('user_session') || 
                       sessionStorage.getItem('isLoggedIn') === 'true';
    
    // If not logged in, redirect to login
    if (!isLoggedIn) {
      window.location.href = '/';
    }
  }
  
  // Run check on page load
  checkAuth();
</script>
